== Text Formatting

_italic text_ +
*_bold and italic_* +
`monospace / inline code` +
#highlighted text# +
[.underline]#underline text# +

....

== Paragraphs & Line Breaks

Normal paragraphs are separated by a blank line.

Use `+` at the end of a line +
to force a line break within the same paragraph.

[%hardbreaks]
Or use the hardbreaks option
to make every newline
a line break automatically.

== Lists

=== Unordered List

* Item 1
* Item 2
** Nested item 2a
** Nested item 2b
*** Deeply nested
* Item 3

=== Ordered List

. First item
. Second item
.. Sub-item a
.. Sub-item b
. Third item

=== Checklist

* [*] Completed task
* [x] Also completed
* [ ] Incomplete task

=== Description List

Term 1:: Definition of term 1
Term 2:: Definition of term 2
Term 3::
Longer definition that
spans multiple lines.

== Code Blocks

=== Inline Code

Use `System.out.println()` to print.

=== Fenced Code Block

[source,python]
----
def hello(name):
    print(f"Hello, {name}!")

hello("World")
----

[source,javascript]
----
function greet(name) {
  console.log(`Hello, ${name}!`);
}
----

=== Code Block with Callouts

[source,yaml]
----
site:
  title: My Docs # <1>
content:
  sources:
    - url: .       # <2>
----
<1> The title shown in the browser tab.
<2> Use `.` for the current directory.

== Admonitions

NOTE: This is a note — general information.

TIP: This is a tip — a helpful suggestion.

IMPORTANT: This is important — pay attention.

WARNING: This is a warning — proceed with caution.

CAUTION: This is a caution — risk of danger.

=== Block Admonition (multi-line)

[WARNING]
====
This is a multi-line warning block.

It can contain *formatted text*, lists, and even code.
====

== Keyboard Shortcuts & UI Macros

Press kbd:[Ctrl+S] to save. +
Press kbd:[Cmd+Shift+R] to hard refresh. +
Select menu:File[Save As...] from the menu. +
Click the btn:[OK] button.
